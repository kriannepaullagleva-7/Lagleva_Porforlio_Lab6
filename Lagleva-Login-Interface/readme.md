# SIMPLE LOGIN , REGISTER, AND DASHBOARD INTERFACE (HTML & CSS)


## Activity Description
This project is an enhancement of the previous **Login Page and Dashboard** activity.  
A new feature was added: **User Registration Page (register.html)**.

The goal of this activity is to practice frontend development using HTML and CSS, while maintaining consistent UI design and proper navigation between pages.


## Page Requirements Implemented

### Register Page (`register.html`)
- UI design is consistent with the Login Page (same colors, fonts, and layout)
- Includes a registration form with the following fields:
  - Full Name
  - Username
  - Email
  - Password
  - Confirm Password
- Includes a **Register button**
- Uses an anchor tag (`<a>`) to redirect users back to:
  - `login.html`
- Includes the text:
  > “Already have an account? Login here”

---

## Pages Included
- `login.html` – User login page
- `register.html` – User registration page
- `dashboard.html` – Dashboard page after login
- `style.css` – Shared stylesheet for consistent design

---

## User Manual (How to Use the Application)

### Step 1: Open the Application
- Download or clone the repository from GitHub
- Open `login.html` using any web browser

### Step 2: Register an Account
- On the Login Page, click **Register**
- Fill out the Registration Form:
  - Full Name
  - Username
  - Email
  - Password
  - Confirm Password
- Click the **Register** button
- You will be redirected back to the Login Page

### Step 3: Login
- Enter your username and password on the Login Page
- Click **Login**
- You will be redirected to the Dashboard Page

### Step 4: View Dashboard
- The Dashboard page confirms successful login
- Navigation is done using anchor tags
